Kekoos World - Unity C#


Free to play online 3D mobile game. It consists on an interactive world that allows users socialize, meet new people, create fully customizable avatar, decorate their private spaces, play games and explore all the different handcrafted spaces. It is still under development and targets all Android and IOS devices.

The main features that I contributed to include:

Online features

  • Backend using .Net and MySql
  • Redis database
  • Multiplayer using Mirror networking.
  • Host Addressables on GoogleCloud to download player clothes, spaces, etc.
  • Added backend API(Net Core) calls to fetch and store data from and to database.

Client features

  • Player camera using cinemachine.
  • Player movement using Unity NavMesh Agent.
  • Interactables system.
  • UI Stack Views Manager.
  • Used Doozy to implement UI.
  • Animations system.
