Unannounced mobile social simulation game

It is still under development and targets all Android and IOS devices. The main features that I have contributed to include:

Online features

  • Backend using .Net and MySql
  • Redis database
  • Multiplayer using Mirror networking.
  • Host Addressables on GoogleCloud to download player clothes, spaces, etc.
  • Added backend API(Net Core) calls to fetch and store data from and to database.
  • Movement and animation online replication

Client features

  • Player camera using cinemachine.
  • Player movement using Unity NavMesh Agent.
  • Interactables system.
  • Avatar editor.
  • Used Doozy to implement UI.
  • Animations system.
